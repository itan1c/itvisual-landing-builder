Глобальні Інтеграції
Налаштуйте автоматичну відправку лідів з форм. При експорті створюються PHP скрипти, які будуть відправляти дані напряму без програміста. Ці налаштування діють для всіх ваших лендінгів.

Налаштування LP CRM

Як знайти API ключ LP CRM?
  1. Зайдіть у вашу адмінку LP CRM (наприклад myshop.lp-crm.biz).
  2. Перейдіть до "Налаштування" -> "API".
  3. Скопіюйте "Secret Key" або згенеруйте новий. Домен - це посилання вашої CRM адмінки.
*Лендінгові форми будуть автоматично ловити поля імені та телефону

Налаштування 7Leads

Як знайти API ключ 7Leads?
  1. Увійдіть у кабінет 7Leads.
  2. Перейдіть у розділ "Мій профіль" -> "Інтеграція по API".
  3. Скопіюйте ваш "Унікальний ключ API" для передачі заявок.

Налаштування SalesDrive

Важливо для SalesDrive!

При експорті лендінгу будуть згенеровані 3 скрипти: my_conf.php (з вашими ключами та товарами), order.php (для відправки) і success.php (для редіректу). Достатньо лише завантажити файли на ваш хостинг.
